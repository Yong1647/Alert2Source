# Outputs directory

Generated artifacts should be written here but are ignored by Git by default.
Typical subdirectories:

- `outputs/kb/`
- `outputs/diagnostic/`
- `outputs/shap/`
- `outputs/reports/`
- `outputs/visual/`
- `outputs/models/`

Large model checkpoints, pickles, crops, and full report JSONs should be stored outside Git or released through Git LFS / Zenodo / release artifacts.
